# AutoJobTracker

AutoJobTracker is a full-stack web application designed to streamline and simplify tracking your job applications. Leveraging the Gmail API and natural language processing (NLP) capabilities, this application automatically creates and updates job application objects, providing users with an organized and efficient platform to manage their job search.

# Key Features:

- **Gmail Integration:** Seamlessly syncs with your Gmail account to analyze incoming emails related to job applications.
- **Natural Language Processing:** Utilizes NLP algorithms to extract relevant information from emails, such as job titles, company names, and application deadlines.
- **Automated Object Creation:** Automatically generates job application objects based on the extracted information, reducing manual data entry and saving time.
- **Customizable Filters:** Allows users to define custom filters and preferences to tailor the application to their specific needs.
- **User-Friendly Interface:** Provides an intuitive and user-friendly interface for managing and tracking job applications, with features such as sorting, filtering, and search functionality.

With AutoJobTracker, users can say goodbye to the hassle of manually organizing job applications and instead focus on finding their next career opportunity.
